# AI睡眠数据检测小程序

## 项目简介

这是一个基于Web技术的AI睡眠数据检测小程序，通过用户拍摄面部和舌头照片，结合健康问卷，生成体质倾向与睡眠数据参考，并提供生活方式建议。

## 功能特点

### 📸 智能拍照识别
- **面部拍照**：拍摄用户面部照片用于面色分析
- **舌头拍照**：拍摄舌头照片用于舌象分析
- **实时预览**：拍照后立即显示照片预览

### 📋 健康问卷系统
- **睡眠质量评估**：了解用户睡眠状况
- **精神状态分析**：评估白天精力水平
- **饮食习惯调查**：分析饮食对体质的影响
- **运动习惯评估**：了解运动对健康的影响

### 🧬 体质分析算法
- **平和体质**：体质平衡，保持良好习惯
- **阴虚体质**：阴液不足，需要滋阴养血
- **气虚体质**：气血不足，需要补气养血
- **痰湿体质**：体内湿气重，需要健脾祛湿

### 💤 个性化睡眠调理
- **睡眠时间建议**：根据体质推荐最佳作息
- **睡眠环境优化**：提供环境改善建议
- **食疗方案**：推荐有助于睡眠的食物
- **运动指导**：适合的运动方式推荐

## 使用方法

### 1. 启动应用
- 直接在浏览器中打开 `index.html` 文件
- 推荐使用Chrome、Safari等现代浏览器

### 2. 开始检测流程
1. **点击"开始智能检测"**：进入拍照页面
2. **拍摄照片**：
   - 点击"拍摄面部"按钮，拍摄清晰的面部照片
   - 点击"拍摄舌头"按钮，伸出舌头拍摄
3. **填写问卷**：回答关于睡眠、精神、饮食、运动的问题
4. **查看结果**：获得体质分析和调理建议

### 3. 分享结果
- 点击"分享结果"按钮，可以将分析报告分享给他人

## 体质判定逻辑

### 评分系统
根据问卷答案计算体质倾向分数：

| 问题选项 | 分数 |
|---------|------|
| 睡眠质量差 | +3分 |
| 精神状态差 | +2分 |
| 饮食油腻 | +2分 |
| 缺乏运动 | +1分 |

### 体质判定标准
- **0-1分**：平和体质
- **2-3分**：痰湿体质
- **4-5分**：气虚体质
- **6分以上**：阴虚体质

## 调理方案详情

### 平和体质
- **睡眠建议**：保持规律作息，晚上10-11点入睡
- **调理方案**：适当运动，营养均衡，定期体检

### 阴虚体质
- **睡眠建议**：滋阴安神，晚上9-10点入睡，喝温牛奶
- **调理方案**：多吃梨、百合、银耳，练习瑜伽；如有不适请线下就医

### 气虚体质
- **睡眠建议**：补气养血，保证充足睡眠，避免过度思虑
- **调理方案**：多吃山药、红枣、黄芪，练习八段锦；如有不适请线下就医

### 痰湿体质
- **睡眠建议**：健脾祛湿，避免睡前过饱，晚餐清淡
- **调理方案**：多吃薏米、赤小豆，增加有氧运动；如有不适请线下就医
